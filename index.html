Welcome to Skylight Pages.

Skylight is the easiest, simplest SDK to create bots in Everybody Edits. It supports quickly connecting to EE via Kongregate, Facebook, or EverybodyEdits; joining levels; receiving information from levels through a simple event system; and performing actions in a level. It is written in C# and supports any .NET language in the 4.0 or higher framework.

static Room R = new Room("PWjThDaofKbEI");
static Bot B = new Bot(R);
static void main()
{
    B.LogIn();
    B.Join();
    Console.Read();
}

// Wow that was too easy.
